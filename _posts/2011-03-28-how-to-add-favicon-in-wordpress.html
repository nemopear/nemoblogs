วันนี้ ได้เพิ่ม Shortcut Icon ซึ่งเขาเรียกว่า "Favicon"
วิธีการไม่ยากเลย แต่ก็งมมาหลายวันเหมือนกัน

กว่าจะรู้ว่าง่ายแสนง่าย..

จึงลองนอกใจ Firefox ..

ไปหา Chrome ...

ไปหา ie9 ...

Icon Nemo หัวจุก มาแล้ว...

แล้ว Firefox ละ ?
..  ยังเฉยชา ..

วิธีการ

  1. สร้าง Icon ขึ้นมา แต่ต้องเป็น *.ico
    --->  ไปแปรรูปจาก *.png หรืออะไรก็ตาม เป็น *.ico ได้ที่ Convertico
    (ตอนนี้ save เป็น favicon.ico)
  2. เอา favicon.ico ไปไว้ใน roots (ตอนนี้ ของเราเป็น htdocs)
  3. ใน header.php ที่Tag <head> ให้เพิ่ม  <link rel="shortcut icon" href="favicon.ico" />
  4. Save --------> เรียบร้อย!!!!
