มีโอกาสได้ประสบปัญหา Image resize ด้วย CSS แล้วภาพมันเบลอ ซึ่งเป็นเฉพาะใน Chrome ด้วย ซึ่งวิธีแก้ก็คือ
เพิ่ม css property 2 ตัวนี้เข้าไป

backface-visibility: hidden;
transform: translateZ(0);
